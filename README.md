A simple, instant web tool to generate 30-second viral video scripts for Shorts, TikTok, or Reels. Turn any boring topic into engaging content.

## Features
- **Dynamic Hook Generation**: Creates attention-grabbing openers.
- **Viral Script Templates**: Uses proven storytelling structures.
- **Auto-CTA**: Generates relevant calls to action.
- **Vibe Suggestions**: Recommends music and on-screen captions.
- **Instant & Lightweight**: No backend, no login, running purely in your browser.

## How to Run Locally
1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. Enter a topic and click "Generate Script".

## Tech Stack
- HTML5
- CSS3 (Responsive & Dark Mode)
- Vanilla JavaScript (No libraries)
